# GCAM-USA Summary

Bundled summary of GCAM-USA.

Use this file after version routing. Keep the exact version route in scope when the user asks about state detail, water markets, or historical GCAM-USA structure.

## What GCAM-USA Adds
- 50 states + D.C. represented as explicit regions.
- State-level energy transformation and end-use demands.
- Inter-state trade of energy goods with state-specific price mark-ups.

## Key Modeling Notes
- Some sectors remain national-scale (e.g., primary fossil resources).
- Biomass supply uses U.S. water basins.
- State-level results can be smoother than real-world lumpy investments.

## Interpretation Cautions
GCAM-USA uses GCAM’s equilibrium and logit-share structure at finer spatial resolution. Results may smooth out discrete investment and retirement decisions.

## Version Notes
- GCAM-USA is most explicitly documented in the later modern families and the bundled `v8.2` baseline.
- Water-market and energy-for-water interactions become more explicit in later modern GCAM-USA releases.
- If the user asks for an exact GCAM-USA path, sector definition, or policy structure in a named release, route to that release before reusing bundled baseline guidance.
